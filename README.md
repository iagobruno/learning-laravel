## Getting started

Clone this repo and run commands in the order below:

```bash
> composer install
> yarn install
> copy .env.example .env # And edit the values
> php artisan key:generate
> php artisan migrate
> php artisan db:seed
> php artisan serve
```

Then open http://localhost:8000/ in your browser.
